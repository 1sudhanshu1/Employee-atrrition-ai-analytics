# PulseHR – Employee Attrition AI Analytics

An AI-powered Employee Attrition Prediction platform that helps HR teams identify employees who are at risk of leaving the organization and provides explainable insights with actionable recommendations.

## Live Demo

https://comfy-paletas-a77bd4.netlify.app/

## Features

- Employee Attrition Prediction
- HR Analytics Dashboard
- Explainable AI (Feature Importance)
- Employee Risk Assessment
- Personalized HR Recommendations
- Interactive Modern UI
- Serverless Prediction API using Netlify Functions
- Fully Responsive Design

---

## Project Overview

Employee attrition is a major challenge for organizations. PulseHR predicts whether an employee is likely to leave based on workplace and employee-related factors.

The system analyzes:

- Age
- Department
- Job Role
- Monthly Income
- Years at Company
- Overtime
- Work-Life Balance
- Stress Level
- Manager Relationship
- Health Score
- Remote Work
- Job Satisfaction
- Workplace Environment

The model calculates:

- Attrition Probability
- Risk Level
- HR Recommendations
- Feature Contribution Analysis

## Installation

Clone the repository

```bash
git clone https://github.com/1sudhanshu1/Employee-attrition-ai-analytics.git
```

Go to the project folder

```bash
cd Employee-attrition-ai-analytics
```

Install dependencies

```bash
npm install
```

Run locally

```bash
npm run dev
```
